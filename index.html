VGG19 with SVM for Resistor Defect Detection using the SolDef AI Dataset

In my current research, I am exploring machine learning approaches to detect tampered or defective PCB components. This project focuses on detecting whether a resistor is defective or benign by analyzing its alignment relative to its soldering pad. Proper alignment is crucial, as misaligned components may signal manufacturing defects or potential tampering. I use a pre-trained VGG19 deep learning model to extract features from images of the resistors. These extracted features, which represent the visual and structural characteristics of the components, are then passed to a Support Vector Machine (SVM) classifier. The SVM determines whether the resistor is classified as defective or non-defective based on these features. The final results need some more work as I only achieve an accuracy of 74 percent. However, by automating defect detection, this project demonstrates the potential for machine learning to improve the efficiency, accuracy, and scalability of hardware inspections.

I implemented a machine learning model, Support Vector Machine, to detect whether a credit card transaction is valid or in valid. The data is pre-processed to favor a machine learning based anomaly detection approach along with labeling for fraudulent transactions. I used scikit libraries to create my machine learning model.

Interactive Chessboard Master

The ICBM is an interactive chessboard that seeks to provide chess players with a way to play against an chess engine opponent on a physical board. With chess's historied origins coming from physical boards, and modern chess being centered increasingly more around digital/online chess, this chessboard seeks to bridge that gap. Using this board, a player can play against another human or an chess engine opponent of variable difficulty. The board includes certain amenities such as a built-in clock, a mechanical arm for the chess engine to interact with the clock, and an array of LED inside the board. The main purpose of the LEDs is to light up certain squares on the board to visually display the chess engine opponent's move. The human player will then move the AI's pieces according to the squares lit up. Finally, the board contains a small LCD screen which functions as a UI for communicating with the AI.

Text to Morse Code

This project objective was to be able to convert plaintext into morse code. I accomplishmented this goal using GNU Radio Compainion. Based in a Linux enviorment, I used python to parse a string into indivdual caharcters that were then labeled to a sequence of dashes and dots. The dashes and dots were then appended to an output vector. This pyhton script was then turned into a python bloack in the gnu radio enviroment. The output vector was then resampled and made into a complex value to be sent over the radio.

Textidence

The React Native application is designed to generate PDF reports based on Short Message Service (SMS) data stored in a local SQLite database. The app retrieves SMS entries upon launch, allowing users to select a specific phone number from a dropdown menu. The selected number is used to filter the SMS data, which is then dynamically incorporated into an HTML template. The HTML content is converted into a PDF file using the RNHTMLtoPDF library. Users are presented with the option to view or share the generated PDF. The app features a user-friendly interface with a title section, a "Generate PDF" button, and a dropdown menu for selecting phone numbers. The application seamlessly interacts with the local database, providing an intuitive and efficient means for users to generate and manage PDF reports based on their SMS data.
